SocialEyes

SocialEyes helps you connect with others through single and group chats while enjoying real-time messaging and notifications for an immersive chatting experience. Features include user profiles, group chats, admin privileges and a typing indicator that keeps the conversations flowing smoothly. User-friendly interface developed using React and the Chakra UI component library. Also implemented secure login and signup features, protecting user data with password encryption, hashing and salting using Bcrypt. Used Socket.IO. for lightning-fast and real-time communication. Be informed and stay in the loop with instant chat notifications.

Codex

Introducing Codex, powered by OpenAI's latest machine learning model. Experience cutting-edge AI-generated responses that surpass expectations. With the homepage built with Vanilla JavaScript using Vite, you can get started effortlessly and dive into the world of AI-powered conversations. By leveraging OpenAI's GPT3 text-davinci-003 model API, our Codex generates unparalleled results. Engage in natural and dynamic conversations with the AI, as it responds with impressive accuracy and fluency. Unlock the potential of AI-driven communication with Codex. Explore the limitless possibilities of intelligent conversations.

Secrets

Introducing Secrets, an anonymous secrets sharing website, where users can freely express themselves without revealing their identities. Share your deepest secrets securely and anonymously with ease. Secrets incorporates robust login and register features with password encryption, hashing, and salting using Bcrypt. Rest assured that your personal information is protected. We've implemented Google OAuth 2.0, allowing users to sign in seamlessly using their Google accounts. Additionally, we've integrated cookies using Passport.js to provide a smooth and personalized user experience.
